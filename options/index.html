Getting started

Switch domain extension adds links to the same page on different domains. It uses simple regular expressions to enable you to match domains and replace sections. This extension was developed primarily for use when you are working on a large project with many test servers.

Example usage

This extension loops though the column and tries to match the current URL, if it finds a match it applies the regexp to ALL the values in the column. Most likely you will have more URLs to generate than matches, in this case add 'ignore' in the matches column.

The above can be quite confusing and hard to manage. To add one to one matches use the Individual matches section.
